# 🎮 Крестики-нолики (Tic-Tac-Toe)

Интерактивная игра «Крестики-нолики» с возможностью:
- начинать новую игру
- возвращаться к любому предыдущему ходу (time travel)
- отслеживать историю ходов


---

## ✨ Возможности

- ✅ Классическая игра 3x3
- 🔄 Перезапуск игры
- 🕒 История ходов
- ⏪ Переход к любому предыдущему состоянию игры
- ⚛️ Управление состоянием через Zustand
- 🎨 Модульные стили (SCSS Modules)
- 🧠 Логика игры отделена от UI

---

## 🛠 Технологии

- **React**
- **TypeScript**
- **Zustand** — управление состоянием
- **SCSS Modules** — стилизация компонентов
- **Vite**
